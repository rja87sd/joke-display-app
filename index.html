Week 6 Day 4 Assignment | API Integration

Instructions:

Part 1: Setup and Documentation

  1. Create a new repository.
  2. Name your repository joke-display-app.
  3. Initialize it with an index.html file, a script.js file, and optionally a style.css file for styling.
  4. Familiarize yourself with the Joke API documentation.
  5. Understand the API endpoints and the structure of the response data.

Part 2: Fetching and Displaying Jokes

  1. In script.js, write a function to make a fetch request to https://v2.jokeapi.dev/joke/Any?safe-mode.
  2. Handle the response and extract the joke data.
  3. Create a card-like element in your HTML to display the joke.
  4. Update the card with a new joke each time you receive a response from the API.
  5. Consider adding a button to fetch a new joke without reloading the page.

Part 3: Testing and Deployment

  1. Ensure the joke is displayed correctly in the card element.
  2. Test the functionality of fetching a new joke with the button.
  3. Commit and push your changes to GitHub.
  4. Deploy your project using GitHub Pages or another hosting service.
  5. Provide the URL link to your GitHub repository and the live deployment as your assignment submission.
